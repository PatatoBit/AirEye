# AirEye

AirEye is an iOS application that estimates Air Quality Index (AQI) using computer vision and machine learning. By analyzing images from your camera or photo gallery, the app provides real-time air quality assessments powered by a custom CoreML model.

## Features

- **Real-time Estimation**: Point your camera to get instant AQI estimates.
- **Gallery Analysis**: Analyze existing photos from your library.
- **Educational Onboarding**: Interactive slides explaining how Computer Vision and ML work.
- **Visual Indicators**: Clear color-coded levels and icons for different air quality conditions.

## Tech Stack

- **Language**: Swift
- **UI Framework**: SwiftUI
- **Machine Learning**: CoreML, Vision
